ATAC-seq SQL Explorer

Use SQL filters to explore TNBC ATAC-seq peak accessibility This tool was built as part of my practice to improve SQL skills, using real data generated from my ATAC-seq TNBC project Check out the Projects page for more details on the analysis

Cancer BLAST

Mini BLAST DNA alignment tool built for practice using k-mer seeding, ungapped alignment, and a simple +1/-1 scoring system. The reference database contains several commonly cancer-related genes
Because computation is performed directly in the browser, sequences longer than 250 bp are not recommended
